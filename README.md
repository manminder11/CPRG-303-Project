Project Description
Mobile Application Project
This project is a mobile application developed using React Native and Expo. The application provides a user-friendly interface to explore various locations and their associated parking spots. The main features of the application include:

Home Screen:

Displays a list of images with corresponding captions.
Allows users to navigate to the search screen by clicking on a search bar.
Search Screen:

Provides a search bar for users to type in their desired location.
Displays a list of previously searched locations (search history).
Allows users to click on a history item or perform a new search to navigate to the parking spots screen.
Parking Spots Screen:

Displays a list of parking spots for the selected location.
Uses a scrollable view to ensure a smooth user experience on both web and mobile devices.
Key Features
Image and Caption Display:

The home screen displays a series of images with captions, providing a visual representation of different locations.
Search Functionality:

Users can search for locations and view their search history. Clicking on a history item or performing a new search navigates the user to the parking spots screen.
Parking Spots Information:

The parking spots screen displays detailed information about parking spots for the selected location. The information is presented in a scrollable view, ensuring accessibility on both web and mobile devices.
